 # Retail Sales Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze retail sales performance across regions, product categories, and cities.

## Key Metrics
- Total Sales: 2.26M
- Total Customers: 793
- Average Sales: 230.77

## Features
- Regional sales analysis
- Category-wise sales comparison
- Monthly sales trends
- Top-performing cities
- Interactive filtering by category, region, and year

## Tools Used
- Power BI
- Excel
